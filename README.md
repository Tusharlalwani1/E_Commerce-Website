# E-Commerce Website 🛒

A modern, responsive **E-Commerce Website** built with **HTML, CSS, and JavaScript**, featuring dynamic product listings, shopping cart functionality, and a smooth user experience.

---

## Features ✨

- Responsive design for desktop, tablet, and mobile  
- Product listing with images, names, prices  
- Add to cart & remove from cart functionality  
- Total price calculation in the cart  
- Smooth animations and interactive UI  

---

## Technologies Used 🛠️

- **HTML5** – Structure & content  
- **CSS3** – Styling & animations  
- **JavaScript** – Interactivity & cart functionality
